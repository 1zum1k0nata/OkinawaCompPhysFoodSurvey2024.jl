# OkinawaCompPhysFoodSurvey2024.jl

みんなで作る WordCloud

このリポジトリは計算物理春の学校 2024 の教材です．

## How to use

- Install Julia

```
$ make build
$ julia --project interactive.jl
$ julia --project formatcheck.jl
$ make test
$ make web
```
